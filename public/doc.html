Welcome

Using the EventStack Partner API is simple. Everything you need to get up and running is here, and for anything not included here, please email apihelp@eventstack.co.

Click the button below to sign up for the API Partner program. You'll receive your key via email, and you are ready to begin making calls.

Partner Key

To use the EventStack Partner API, please request a key below. The key will be emailed to you. This key needs to be included in the header of every API request made. Requests without a valid Key included will be returned an HTTP 401 Unauthorized error.

Rate Limits

There is a rate limit of 100 Partner API requests per minute. Requests submitted in excess of this will return an HTTP 429 Error "Too Many Requests".

Events

The Events API can be used to retrieve a listing of local events for a metro area, ordered by distance from a specified point.

events/today?lat={lat}&lng={lng}&page={page}
latThe latitude value of the point the desired events should be centered around. The Events API will return up to 100 events in groups of 20 events, sorted by distance.
lngThe longitude value of the point the desired events should be centered around. The Events API will return up to 100 events in groups of 20 events, sorted by distance.
pageOptional, default 0. Controls the group number of 20 events. For example, page=0 returns events 0-20; page=1 returns events 21-40.

Response Codes

Possible response and error codes:

200GET request received, response successfully returned.
202POST received.
401Unauthorized error. Partner key is not valid.
429Rate limit error. Rate limit was exceeded.
